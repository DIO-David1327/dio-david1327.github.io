Fonctionnalités de DIO-TOOLS-David1327

Vous trouverez ici la description détaillée des fonctions du DIO-TOOLS-David1327.

Dio-tools est une petite extension pour le jeu par navigateur grepolis. Il propose, entre autres, quelques affichages (compteur de Birèmes, force d'unité...), une boîte de sélection de smileys pour les champs de saisie de texte, des options d'échange et quelques modifications de mise en page. Presque chaque fonctionnalité peut être désactivée dans les paramètres.

En ouvrant la page de paramètres "DIO-TOOLS-David1327", cela se fait avec le bouton de paramètres sur la barre latérale droite de la page principale du jeu, nous pouvons voir l'existence de 9 onglets différents, chacun fournissant plusieurs modules qui peuvent être activés ou désactivé. selon votre besoin.

Pour voir les fonctionnalités détaillées (localisation, fonction…) cliquez sur l'angle désiré
