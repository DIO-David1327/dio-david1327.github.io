Features of DIO-TOOLS-David1327

Here you will find the detailed description of the functions of the DIO-TOOLS-David1327 script.

Dio-tools is a small extension for the browser game grepolis. It offers, among other things, some displays (Biremes counter, unit strength...), a smiley selection box for text input fields, trade options and some changes to the layout. Almost each feature can be deactivated in the settings.

Opening the "DIO-TOOLS-David1327" settings page, this is done with the settings button on the right sidebar of the game's main page, we can see the existence of 9 different tabs, each of which provides several modules that can be enabled or disabled. according to your need.

To see the detailed functionalities (location, function, etc.), click on the desired angle
