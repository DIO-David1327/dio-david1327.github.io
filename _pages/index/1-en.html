DIO-TOOLS-David1327 what is this?

It is a small extension for the browser game Grepolis that integrates with the internet browser.

They allow to have additional options (counter, displays, smilies, trade options, changes to the layout). A powerful tool, which improves the gameplay so much that nowadays many players cannot imagine the game without it.
